# Witaj w Storybook

Storybook pomaga budować komponenty UI w oderwaniu od logiki biznesowej, danych i kontekstu aplikacji.
W Dare IT React Challenge użyjesz go do pracy nad pierwszym zadaniem, tak aby móc przygotować komponenty w oderwaniu od całości aplikacji.
Twoje rozwiązanie zostanie zastosowane w kolejnych zadaniach :)

<div className="subheading">Jak z niego korzystać?</div>
<ul>
    <li>Wybierz z menu po lewej stronie jedną z zakładek</li>
    <li>Zobczysz komponent wraz z możliwymy parametrami w sekcji "Controls"</li>
    <li>Możesz edytować kod komponentu znajdującego się w `client/src/ui`</li>
</ul>
